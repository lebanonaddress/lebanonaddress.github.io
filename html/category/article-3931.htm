RestTemplate中文问号?

首页 / 文章分类

问题描述

最近项目里面，要调外部接口发送短信，发现用Postman可以正常调接口收短信，但是代码里面用RestTemplate调接口，接口通的，但是却一直收不到短信，一开始以为是请求的头部有问题，代码里面打印出来请求的头部、body，用Postman请求接口就可以收到短信，没办法只能请对方协助排查，发现原来RestTemplate调接口发送过去的数据，由于编码问题，传过去都变成了问号?。

解决方法

设置RestTemplate编码为UTF-8

RestTemplate restTemplate = new RestTemplate(); // 设置restTemplate编码为utf-8 restTemplate.getMessageConverters().set(1, new StringHttpMessageConverter(StandardCharsets.UTF_8));  HttpHeaders headers = new HttpHeaders(); headers.setContentType(MediaType.parseMediaType(MediaType.APPLICATION_JSON_UTF8_VALUE)); headers.add("Accept", MediaType.APPLICATION_JSON_UTF8_VALUE); headers.set("Authorization", token);  HttpEntity<String> httpEntity = new HttpEntity<>(pushData, headers); ResponseEntity<String> response = restTemplate.exchange(smsAddress, HttpMethod.POST, httpEntity,String.class); if (HttpStatus.OK.equals(response.getStatusCode())) {     value = response.getBody(); } else {     log.info("失败，状态码：{}", response.getStatusCode()); }

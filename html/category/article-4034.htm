1、vscode离线安装扩展

进入vscode官网：https://code.visualstudio.com/，除了能下载vscode安装报以外，还能下载扩展工具.vsix文件。

搜索需要的扩展工具

下载扩展.vsix文件

在vscode选择安装离线扩展，如下图

选择之后导入即可；安装成功之后在enabled一栏即可看到安装的扩展；

2、vscode在线安装扩展

选择扩展工具一栏，搜索扩展名称，直接install即可；

安装完毕之后若要卸载或禁用该扩展，直接选择已安装扩展禁用即可；

3、常用扩展工具推荐

我们在开发时选择一些好用的插件，能为我们提高不少开发效率；推荐以下几个常用插件：

  • Auto Close Tag ：自动生成关闭标签，使用前端绝大部分标记语言，推荐安装。
  • Bracket Pair Colorizer：给括号加上不同颜色，js开发推荐，无论小括号还是大括号，相同颜色进行匹配；推荐安装。
  • Prettier - Code formatter： 代码格式化工具，可以格式化绝大多数语言，美化代码；
  • Git Project Manager：具有强大git集成以及管理功能，按需安装。
  • Turbo Console Log： console打印管理，适用前端经常需要打印日志的场景，推荐安装。
  • Vetur：vue开发者必备，vscode中集成vue开发工具，按需安装；
